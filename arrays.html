Arrays

Los arrays o las listas es la estructura de datos más común, podría decirse que es la que más vamos a usar comúnmente en nuestros proyectos de programación. Y está conformado por un valor y su respectivo índice.

Tipos de array

Los arrays se dividen comúnmente en dos tipos:

  • Array dinámico: Un array dinámico es un array que se puede modificar después de haber sido creado, este tipo de array nos permitiría agregar o eliminar elementos.
  • Array estático: A diferencia del array dinámico, este array no puede modificarse (no podemos modificar su tamaño, agregando o eliminando elementos).

¿Cómo podríamos crear un array desde 0 con JavaScript?

Para poder crear un array desde 0 hay que tener en cuenta todo lo que compone al array, desde que cada elemento debe tener su respectivo índice que va a incrementar una vez agreguemos más elementos y va a decrementar cada vez que eliminemos alguno.

Para crear un ejemplo de como funcionaría un array lo crearemos con el paradigma de la programación orientada a objetos, haremos uso de un objeto como lugar de almacenamiento de los datos y de una propiedad length que será la encargada de guardar el tamaño del array.

                    
class array {
    constructor() {
        this.length = 0; // Iniciamos en 0
        this.data = {};
    }
}
